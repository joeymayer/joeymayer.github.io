Atanasoff-Berry Computer Emulator

Download the abc-emulator jar here and follow the video along.

Download JAR

Overview

Full emulation of the of the Atanasoff-Berry Computer (ABC) using Java and Swing as the UI. Used to solve Linear Systems of Equations using Atanasoff's Modified Gaussian Elimination.

Components

Input/Output

Decimal and Binary cards are both used for I/O. They are represented as CSV files and stored in the resources bin.
